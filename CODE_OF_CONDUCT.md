# Code of Conduct

## Our Pledge
We pledge to create a welcoming and inclusive environment.

## Our Standards
Respectful communication and constructive feedback are expected.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the rules.

## Scope
This Code of Conduct applies to all project spaces.

## Enforcement
Instances of abusive behavior may be reported to maintainers.

## Enforcement Guidelines
Maintainers will review complaints and take appropriate action.

This Code of Conduct is adapted from Contributor Covenant.
